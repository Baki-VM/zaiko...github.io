2.1 Componentes lógicos y aplicaciones informáticas

Los componentes lógicos son la parte intangible del sistema informático: programas, aplicaciones, sistemas operativos y todo aquello que hace funcionar el hardware y nos permite realizar tareas.

Ejemplos incluyen desde Windows y Linux, hasta las apps que usas para todo día a día.

Tipo Descripción
Sistema operativo Software base que gestiona el hardware y recursos del equipo.
Aplicaciones Programas diseñados para tareas específicas (navegadores, juegos, utilidades).
Utilidades Herramientas para optimizar, proteger o mejorar el sistema.

2.2 Mantenimiento en Software

El mantenimiento es vital para que el software siga funcionando bien, corrigiendo errores y adaptándose a nuevos entornos.

Sin mantenimiento, las aplicaciones se vuelven lentas, inseguras o simplemente obsoletas.

Tipo de mantenimiento Descripción
Correctivo Corrección de fallos y errores.
Adaptativo Ajustes por cambios en hardware o software.
Perfectivo Mejoras y optimizaciones.
Preventivo Prevención de futuros problemas.

2.4 Software libre

El software libre es como un libro abierto: cualquiera puede usarlo, modificarlo y compartirlo. Es colaboración y libertad digital en su máxima expresión.

Grandes ejemplos incluyen Linux, Firefox y LibreOffice.

Leer más

2.5 Software propietario

Este software es cerrado y protegido: solo el creador controla su uso y distribución. Por eso, sus licencias son estrictas.

Ejemplos clásicos son Windows, Adobe Photoshop y Microsoft Office.
