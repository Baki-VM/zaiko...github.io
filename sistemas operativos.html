🖥️ 1.18 Sistemas Operativos

📌 ¿Qué es un Sistema Operativo?

Un Sistema Operativo (SO) es el alma digital de tu computadora: gestiona el hardware, optimiza los recursos y te conecta con la máquina a través de interfaces visuales o comandos.

🛠️ Funciones Principales

  • 🧠 Administración de procesos: organiza la ejecución de programas en segundo plano y en primer plano.
  • 💾 Gestión de memoria: asigna espacio para aplicaciones y libera cuando ya no es necesario.
  • 🖨️ Control de periféricos: impresoras, teclados, ratones, ¡todo conectado y sincronizado!
  • 🗂️ Gestión de archivos: organiza carpetas, accesos y permisos.
  • 🔒 Seguridad: protege tu sistema y datos ante amenazas.
  • 🖥️ Interfaz de usuario: la puerta visual y amigable al mundo digital.

📊 Tipos de Sistemas Operativos

Tipo Características Ejemplos
De escritorio Interfaz visual completa, multitarea y acceso a software diverso Windows, macOS, Linux
De servidor Optimizado para redes, bases de datos y seguridad Windows Server, Red Hat, Ubuntu Server
Embarcados Diseñados para dispositivos con funciones específicas FreeRTOS, VxWorks
Móviles Interfaz táctil, consumo eficiente de batería Android, iOS

📈 Evolución de los Sistemas Operativos

  • 1950: Sistemas batch o por lotes - Sin interacción directa, solo procesamiento secuencial.
  • 1960: Multiprogramación - Capacidad de ejecutar varios programas a la vez.
  • 1970: UNIX - Portabilidad y base de muchas arquitecturas modernas.
  • 1980: GUI - Aparición de las interfaces gráficas (Apple Lisa, Macintosh).
  • 1990: Windows se impone, democratizando la informática personal.
  • 2000: Mobile OS - Android e iOS dominan la era de smartphones.
  • 2010-Actualidad: Nube, virtualización e IA en sistemas operativos.
