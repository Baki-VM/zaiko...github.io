Instalación Dual Boot: Windows 11 & Ubuntu

Guía completa paso a paso con imágenes, tablas detalladas y consejos profesionales para instalar ambos sistemas operativos en una sola laptop

Introducción

La instalación dual boot te permite tener dos sistemas operativos en el mismo equipo, eligiendo cuál usar al iniciar. Esta guía cubre la instalación de Windows 11 y Ubuntu 22.04 LTS, el método más estable y compatible para desarrolladores y usuarios avanzados.

Nota importante: Este proceso eliminará todos los datos de tu disco duro. Realiza una copia de seguridad completa antes de comenzar.

Requisitos Previos

Hardware

  • USB de 8GB mínimo (2 unidades)
  • Laptop con UEFI (no Legacy BIOS)
  • Espacio en disco: Mínimo 150GB (recomendado 256GB+)
  • CPU de 64-bit con 2+ núcleos
  • 4GB RAM (8GB recomendado)

Software

  • Windows 11 ISO: Media Creation Tool
  • Ubuntu 22.04 LTS ISO
  • Rufus (grabador USB)
  • GParted Live ISO (opcional)

Configuración BIOS/UEFI

  • Secure Boot: Desactivado
  • Modo de arranque: UEFI (no Legacy)
  • Fast Boot: Desactivado
  • TPM: Activado (requerido para Windows 11)

Preparar USB de Instalación

Crear USB de Windows 11

  1. Descarga la herramienta Media Creation Tool de Microsoft
  2. Ejecútala y selecciona "Crear medios de instalación"
  3. Elige idioma, edición y arquitectura (64-bit)
  4. Selecciona "Disco USB"
  5. Espera a que finalice el proceso (20-40 minutos)

Alternativa con Rufus: Si usas Rufus, configura:
- Esquema: GPT
- Sistema destino: UEFI
- Sistema de archivos: NTFS

Crear USB de Ubuntu

  1. Descarga la ISO de Ubuntu Desktop
  2. Abre Rufus y selecciona tu USB
  3. Haz clic en "Seleccionar" y busca la ISO de Ubuntu
  4. Configura:
    • Esquema de partición: GPT
    • Sistema destino: UEFI
    • Sistema de archivos: FAT32
  5. Haz clic en "Empezar"
  6. Espera a que finalice (10-20 minutos)

Particionamiento del Disco

El esquema de particiones recomendado para instalación dual:

Particion Tamaño Tipo Sistema de Archivos Notas
EFI 500 MB Primaria FAT32 Obligatoria para UEFI
Windows 100 GB Primaria NTFS Sistema + programas
Ubuntu / 40 GB Lógica EXT4 Sistema Linux
Ubuntu /home 60+ GB Lógica EXT4 Datos de usuario
Swap = RAM Lógica Área de intercambio Ej: 8 GB (si RAM < 8 GB)

Proceso de Particionamiento

  1. Inicia desde el USB de Windows
  2. En "¿Dónde desea instalar Windows?" haz clic en "Opciones de unidad"
  3. Elimina todas las particiones existentes (¡copia de seguridad primero!)
  4. Crea nueva partición:
    • Tamaño: 500 MB → se creará automáticamente como EFI
  5. Crea partición principal para Windows:
    • Tamaño: 100 GB
    • Sistema de archivos: NTFS
  6. No crees particiones para Ubuntu todavía

Precaución: Deja espacio sin asignar para Ubuntu (mínimo 50GB, recomendado 100GB+). Ubuntu creará sus particiones durante su instalación.

mostrando partición EFI y NTFS

Instalar Windows 11

  1. Inicia desde el USB de Windows (accede al menú de arranque con F12, F10 o Esc según tu laptop)
  2. Configura idioma, hora y método de entrada
  3. Haz clic en "Instalar ahora"
  4. Introduce tu clave de producto o haz clic en "No tengo clave"
  5. Selecciona la edición de Windows 11 (Home/Pro)
  6. Acepta los términos de licencia
  7. Elige "Personalizada: Instalar solo Windows"
  8. Selecciona la partición de 100 GB que creaste
  9. Haz clic en "Siguiente" y espera a que termine la instalación
  10. Configura tu usuario y preferencias iniciales

Después de instalar Windows, inícialo completamente al menos una vez para asegurar que todo funciona correctamente antes de instalar Ubuntu.

Imagen: Selección de partición durante
instalación de Windows 11

Instalar Ubuntu

  1. Inicia desde el USB de Ubuntu (menú de arranque)
  2. Selecciona "Probar Ubuntu" o "Instalar Ubuntu"
  3. Configura tu idioma y distribución de teclado
  4. En "Tipo de instalación":
    • ¡Clave! Elige "Instalar junto a Windows Boot Manager"
  5. El instalador mostrará el espacio libre que dejaste para Ubuntu
  6. Si prefieres control manual:
    • Selecciona "Algo más"
    • Crea partición raíz (/): 40 GB, EXT4
    • Crea partición swap: 8 GB, tipo "área de intercambio"
    • Crea partición /home: resto del espacio, EXT4
    • Dispositivo para el cargador de arranque: La partición EFI (ej: /dev/sda1)
  7. Selecciona tu zona horaria
  8. Crea tu usuario y contraseña
  9. Haz clic en "Instalar" y espera 15-30 minutos
  10. Reinicia cuando se complete

El gestor de arranque GRUB se instalará automáticamente y detectará Windows 11. Al iniciar tu laptop verás un menú para elegir entre Ubuntu y Windows.

Configuración Final

GRUB - Gestor de Arranque

Tras instalar Ubuntu, el gestor de arranque GRUB detectará Windows 11 automáticamente. Al iniciar tu laptop verás:

Imagen: Menú GRUB mostrando opciones
para Ubuntu y Windows 11

Si Windows no aparece en el menú:

Abre Terminal en Ubuntu y ejecuta:

sudo update-grub

Reinicia y Windows debería aparecer en el menú.

Sincronización de Hora

Windows y Linux manejan el hardware clock de forma diferente. Para solucionar diferencias de hora:

En Ubuntu ejecuta:

timedatectl set-local-rtc 1 --adjust-system-clock

Problemas Comunes y Soluciones

Problema Solución
No arranca GRUB Accede a UEFI y cambia el orden de arranque para poner Ubuntu primero. Ejecuta Boot-Repair desde USB Live.
Secure Boot bloquea Ubuntu Desactívalo en UEFI o instala controladores firmados: sudo apt install shim-signed
Windows no aparece en GRUB Ejecuta sudo update-grub en Ubuntu. Si persiste, reinstala GRUB.
Error de espacio en disco Usa GParted Live USB para redimensionar particiones sin perder datos.
Problemas con WiFi o gráficos Actualiza drivers: sudo apt update && sudo apt upgrade. Instala drivers propietarios en "Software & Updates".

Boot-Repair: Herramienta esencial para resolver problemas de arranque. Instálala en Ubuntu con: sudo apt install boot-repair y ejecútala.

Conclusión

Con esta guía has aprendido a instalar Windows 11 y Ubuntu en la misma laptop en modo dual boot. Ahora puedes disfrutar de lo mejor de ambos sistemas operativos:

  • Windows 11: Para aplicaciones de oficina, juegos y software exclusivo
  • Ubuntu: Para desarrollo, servidores, y herramientas de código abierto

Recuerda mantener ambos sistemas actualizados y realizar copias de seguridad periódicas de tus datos importantes.

¡Felicidades! Ahora tienes un sistema dual boot completamente funcional. Al iniciar tu laptop podrás elegir entre Windows 11 y Ubuntu según tus necesidades.
